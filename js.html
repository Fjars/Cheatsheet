Comparison Operators

Syntax

Name

Description

<

Less than

Returns true if something is less than.

>

Greater than

Returns true if something is greater than.

<=

Less than or equal to.

Returns true if something is less than or equal to.

>=

Greater than or equal to.

Returns true if something is greater than or equal to.

===

Is equal to

True if statements are equal.

!==

Is not equal to

Returns true if statements are not equal.

Logical Operators

Syntax

Name

Description

&&

The and operator

Checking if two things are true and returns true or false.

||

The or operator

Returns true if at least one statement is true.

!

The not operator

Takes true and returns false or the opposite.
